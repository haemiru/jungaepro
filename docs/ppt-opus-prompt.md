## 기술 요구사항 (python-pptx)

- `docs/generate-report.py`의 `make_presentation`, `blank`, `set_bg`, `add_textbox` 등 헬퍼 구조를 참고해 동일한 코드 스타일로 작성
- 이미지 삽입 전 `Path.exists()` 체크, 없으면 파일명을 표기한 회색 placeholder 사각형으로 대체 (스크립트가 절대 죽지 않게)
- 스크린샷 비율 유지: PIL 없이 python-pptx만으로 처리하려면 width만 지정하고 height는 자동 계산되도록 `add_picture(path, left, top, width=...)` 사용
- 슬라이드 하단에 페이지 번호 + "중개프로 · jungaepro.com" 푸터 (표지 제외)
- 실행: `py docs/generate-product-intro.py` (python-pptx >= 1.0)

- 13번 슬라이드의 QR코드는 jungaepro.com 가입 페이지 링크로 실제 생성해 교체.
- 14번 슬라이드의 문의 이메일/전화는 발송 전 실제 값으로 기입.
